## 🎯 개발 목표

### 주요 목표
1. 기존 고객 정보 관리를 서비스 이용 내역과 통합
2. 패키지 사용률 실시간 추적 및 관리
3. AI 기반 개인화 서비스 추천 시스템 구축
4. 고객 분석 대시보드 제공

### 성공 지표
- 고객 정보 조회 시간 50% 단축
- 패키지 갱신율 70% → 85%
- 시스템 응답 시간 500ms 이내

## 🛠️ 기술 스택 및 도구

### 백엔드
- **Framework**: FastAPI 0.115.5
- **Database**: PostgreSQL 17 + SQLAlchemy 2.0
- **Cache**: Redis (선택적)
- **Task Queue**: Celery (백그라운드 작업용)

### 프론트엔드
- **Framework**: React 19 + TypeScript 5.4
- **상태 관리**: Zustand
- **UI 라이브러리**: Ant Design 5.x
- **차트**: Recharts
- **API 클라이언트**: Axios + React Query

### 개발 도구
- **버전 관리**: Git + GitHub
- **CI/CD**: GitHub Actions
- **테스트**: Jest, Pytest
- **문서화**: Swagger UI

## 🚀 개발 프로세스

### 1. 개발 환경 설정
```bash
# 백엔드
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# 프론트엔드
cd frontend
npm install
```

### 2. 브랜치 전략
- `main`: 프로덕션
